# QR Code Generator

A modern QR code generator built with HTML, CSS, Bootstrap, and JavaScript for quick sharing, downloading, and live preview creation.

## Live Demo

- Website: https://www.creativesoft.shop
- See live QR code generator: https://www.creativesoft.shop/qr-code-generator

## Company

Creative Vsion Information Technology

## Features

- Generate QR codes from a website link or any custom text.
- Adjust QR size, foreground color, background color, and error correction.
- Download the QR code as a PNG image.
- Copy the source text with one click.
- Responsive layout powered by Bootstrap for desktop and mobile screens.
- Clean glass-style design that is ready for GitHub Pages or any static hosting.

## Built With

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- QRCode.js via CDN

## How To Use

1. Open the project in a browser by loading `index.html`.
2. Paste a URL, message, or business detail into the text box.
3. Choose the QR style options you want.
4. Click Generate QR and download the result when you are happy with it.

## Project Files

- `index.html` - page structure and Bootstrap layout
- `styles.css` - custom visual styling and responsive design
- `script.js` - QR generation, download, reset, and copy logic
